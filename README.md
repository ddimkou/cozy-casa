# Cozy-Casa Project

- React Typescript
- Vite
- Tailwind
- HeroIcons
- React CountUp / react-visibility-sensor
- ESLint

### Link to project: https://cozy-casa.netlify.app/

### Getting Started.

Make sure you have Node.js and npm

    git clone https://github.com/ddimkou/cozy-casa.git

    cd supercar-rent

    npm install

    npm run dev
